5.# Notification Delivery System (Constructor Injection)

This project demonstrates **Constructor Injection** in Spring Core by implementing a simple notification delivery system.

## Features
- Loose coupling using constructor injection
- Multiple notification implementations (Email, SMS, etc.)
- Easy to extend and test
- Clean separation of responsibilities

## Concepts Used
- Spring Core
- Dependency Injection
- Constructor Injection
- Interfaces & Implementations

## Purpose
This project is created to understand **why constructor injection is preferred over field/setter injection** in real-world Spring applications.
